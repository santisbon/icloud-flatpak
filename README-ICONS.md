# Icon Download Instructions

Download the following 100x100 PNG icons from https://icons8.com/ (free license):

1. **iCloud Mail** - Search "mail" or "email" → Save as `icons/icloud-mail.png`
2. **iCloud Drive** - Search "cloud storage" or "drive" → Save as `icons/icloud-drive.png`
3. **iCloud Calendar** - Search "calendar" → Save as `icons/icloud-calendar.png`
4. **iCloud Contacts** - Search "contacts" or "address book" → Save as `icons/icloud-contacts.png`
5. **iCloud Photos** - Search "photos" or "gallery" → Save as `icons/icloud-photos.png`
6. **iCloud Notes** - Search "notes" or "notepad" → Save as `icons/icloud-notes.png`
7. **iCloud Reminders** - Search "reminders" or "tasks" → Save as `icons/icloud-reminders.png`
8. **iCloud Find My** - Search "location" or "find" → Save as `icons/icloud-find.png`

## How to Download from Icons8

1. Visit https://icons8.com/
2. Search for the icon name
3. Select an appropriate icon
4. Choose "100px" size
5. Choose "PNG" format
6. Download with free license (requires attribution in your app)
7. Save to the `icons/` directory with the name specified above
